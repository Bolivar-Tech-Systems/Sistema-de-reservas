# Sistema de Reservas

App de reservas desarrollada con Flutter y FastAPI.

## Tecnologías
- **Frontend:** Flutter
- **Backend:** FastAPI + Python
- **Base de datos:** PostgreSQL

## Requisitos
- Flutter SDK
- Python 3.12+
- PostgreSQL

## Instalación

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Flutter
```bash
flutter pub get
flutter run
```
